Text Formatting

Text Formatting adalah pemberian format yang mengkhusus pada text. Elemen format dirancang untuk menampilkan jenis teks khusus.

Contohnya:
<b> : Ini untuk teks tebal
<strong> : Ini untuk teks penting
<i> : Ini untuk teks miring
<em> : Ini untuk teks penekanan teks
<mark> : Ini untuk menandai teks
<small> : Ini untuk teks lebih kecil
<del> : Ini untuk teks dicoret
<ins> : Ini untuk menyisipkan teks
<sub> : Ini untuk teks subskrip
<sup> : Ini untuk teks superskrip
