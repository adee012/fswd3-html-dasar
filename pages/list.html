List Dalam HtML

List adalah fungsi dalam HTML yang digunakan untuk menampilkan daftaran yang berurutan ke bawah

Ada list terurut atau ordered list dan ada list yang tidak terurut atau unordered list

  1. Ordered List atau list yang terurut

    2. List terurut menggunakan <ol> dan untuk membuat list itemnya menggunakan <li>

    Contoh penggunaannya:
    1. Presiden Soekarno (1945-1967)
    2. Presiden Soeharto (1967-1998)
    3. BJ Habibie (1998-1999)
    4. Abdurrahman Wahid (1999-2001)
    5. Megawati Soekarno Putri. (2001-2004)
    6. Susilo Bambang Yudhoyono (2004-2014)
    7. Joko Widodo (2014-Sekarang)


  2. Unordered List atau list yang tidak terurut

    3. List tidak terurut menggunakan <ul> dan untuk membuat list itemnya menggunakan <li>
